## 1. Create Dataset Using Label Studio

Tutorial video:
[https://www.youtube.com/watch?v=r0RspiLG260](https://www.youtube.com/watch?v=r0RspiLG260)

### Steps

1. Install Label Studio

   ```
   pip install label-studio
   ```

2. Start Label Studio

   ```
   label-studio
   ```

3. Create a new project â†’ Upload extracted frames.

4. Choose **Object Detection (Bounding Boxes)**.

5. Label exercises (plank, squat, pushup, etc.).

6. Export annotations in **YOLO format** and place the `data.zip` inside the project.

---

## 2. Extract Frames From Videos

Run this script before labeling:

```
python video.py
```

This extracts frames from:

* video1.mp4
* video2.mp4
* workout1.mov
* workout2.mov

---

## 3. Train YOLO Model

1. Unzip the dataset generated by Label Studio.
2. Train YOLO using:

```
yolo task=detect mode=train data=data.yaml model=yolov8n.pt epochs=50 imgsz=640
```

This creates your model:

```
my_model.pt
```

---

## 4. Run Inference on Videos

Use your trained model on any video:

```
python yolo_detect.py --model my_model.pt --source workout2.mov
```

Optional flags:

* `--show` = display prediction window
* `--save` = save output video
